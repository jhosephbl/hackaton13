¿Por qué evitar los callbacks en las llamadas asíncronas?
Para evitar el callback Hell o anidamiento de callbacks.

¿Qué es una promesa?
Una promesa es un objeto que sirve para retornar un valor correcto o incorrecto, esto lo podemos hacer usando then o catch

¿Qué es el callback hell?
Consiste en multiples callbacks anidados que hacen la lectura mas dificil
